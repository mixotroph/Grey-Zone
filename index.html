Welcome to Grey Zone

Grey Zone ist ein rogue-like Dungeon-crawler, der in Java geschrieben wurde. Er ist Rahmen eines Softwarepraktikums am Instituts für Informatik der Freien Universität Berlin enstanden (Projekthomepage). Das Projekt kann auf Github hier verfolgt werden.

(Startscreen)

Story

Killed in a key-experiment, researching evil, Dr.Eisenhower has to make a descision: Fighting his way through his labratory or hell. Trying to save his soul and research, he learns of the relativity of boarders between good and bad.

Instalation

  1. Lade die Daten mit git von der Projektseite: git clone https://github.com/mixotroph/Grey-Zone.git greyzone

  2. Wechsel mit deinem Terminalclient in das neue Projektverzeichnis: cd greyzone/src

  3. Kompiliere das Projekt: javac greyzone/greyzone.java

  4. Starte das Programm: java greyzone/greyzone

Steuerung

  • w - nach oben gehen
  • s - nach unten gehen
  • a - nach links gehen
  • d - nach rechts gehen
